# 🧠 Projetos de Machine Learning e Engenharia de Processos

Bem-vindo(a) ao meu repositório! 🚀  
Aqui compartilho projetos e análises que desenvolvi, unindo **Engenharia Química** e **Machine Learning** para otimizar processos, prever falhas e gerar insights a partir de dados.

---

## 📂 Conteúdo do Repositório

| Projeto | Descrição | Tecnologias |
|--------|-----------|------------|
| 🔹 **Predição de Falhas de Máquinas** | Modelos de classificação para prever falhas com **LazyPredict**, **VAE** e técnicas de balanceamento de dados | Python, Pandas, Scikit-learn, TensorFlow |
| 🔹 **Análise de Composição Química em Amostras de Cerâmica** | Modelos de classificação usando pacote de **AutoML** e **Autoencoder Esparso** para fugir do overfitting  | Python, AutoML, Matplotlib, TensorFlow |
| 🔹 **Previsão de Propriedades Físico-Químicas de Polímeros** | Modelagem baseada em **SMILES** e embeddings moleculares com redes neurais variacionais | Python, RDKit, PyTorch | (em construção)
